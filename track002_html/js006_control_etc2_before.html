008. 유용한 기능 - STEP1. 빈칸검사

STEP1. 빈칸검사 

    1. #blank 버튼 클릭 시 #blank_target 입력값이 비어있다면 '빈칸입니다. 확인해주세요' 알림창을 띄운다.
    2. 빈칸일 경우 focus를 input으로 이동시킨다.

EX- 빈칸유효성검사

      Q1. #plus_form 안에 있는 input type="text"들의 배경색을 하얀색으로 변경
      Q2. #divide 버튼을 클릭하면 빈칸검사
      Q3. 빈칸이 아니라면 소수점 둘째자리 나누기 결과를 출력함.
JAVASCRIPT DIVIDE




008. 유용한 기능 - STEP2. css 적용

STEP2. css 적용 

    1. #fn_css 버튼 클릭 시 btn-success 클래스가 있다면 제거한다.
    2. #fn_css 버튼 클릭 시 btn-success 클래스가 없다면 추가한다.
    3. 또는 classList.toggle()을 활용하여 btn-success 클래스를 토글한다.

EX

    1.  #fn_css_test 버튼에  클릭 시  btn-outline-success 가 있다면 btn-outline-danger로
    2.  #fn_css_test 버튼에  클릭 시  btn-outline-danger  가 있다면 btn-outline-success로




008. 유용한 기능 - STEP3. 속성 적용

STEP3. 속성 적용 

    1.  #attr_basic 이미지를 클릭 시 현재 src 속성을 알림창으로 출력한다.
    2.  클릭 후 src 속성을 img/b02.jpg 로 변경한다.  (https://picsum.photos/200/200?random=2)
