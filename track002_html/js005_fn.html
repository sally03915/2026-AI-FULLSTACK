  1. 문제: MOVE & COLOR 박스 만들기
  아래 조건을 만족하는 HTML + JavaScript 코드를 작성하시오.
  - 화면 중앙에 MOVE & COLOR라는 글자가 들어 있는 박스를 만든다.
  - 박스는 Bootstrap의 bg-warning 색상을 기본으로 가진다.
  - 박스 안의 글자는 흰색(text-white)이며 가운데 정렬한다.

  2. 박스 아래에 세 개의 버튼을 만든다.
  - MOVE LEFT 버튼: 박스를 왼쪽으로 50px씩 이동시킨다.
  - MOVE RIGHT 버튼: 박스를 오른쪽으로 50px씩 이동시킨다.
  - CHANGE COLOR 버튼: 박스의 배경색을 순서대로 바꾼다.
  (bg-warning → bg-success → bg-danger → bg-info → bg-primary → 다시 bg-warning)

  버튼 클릭 시 기본 동작(링크 이동)은 막아야 한다. (e.preventDefault() 사용)

  
  3. 출력 예시 화면:
  - 초기 화면: 노란색 박스(bg-warning)가 중앙에 위치한다.
  - MOVE LEFT 클릭 시 박스가 왼쪽으로 이동한다.
  - MOVE RIGHT 클릭 시 박스가 오른쪽으로 이동한다.
  - CHANGE COLOR 클릭 시 박스의 색상이 순서대로 바뀐다.
