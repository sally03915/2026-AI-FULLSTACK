예제 1: position static

STATIC

기본값으로, 특별한 위치 지정 없이 문서 흐름에 따라 배치됩니다.

예제 2: position relative

RELATIVE

자기 자신을 기준으로 top/left/right/bottom 값을 적용합니다.

예제 3: position absolute

ABSOLUTE

부모 요소(position: relative) 기준으로 위치가 결정됩니다.

예제 4: position fixed

FIXED

브라우저 화면(viewport)을 기준으로 항상 같은 위치에 고정됩니다.

예제 5: position sticky

STICKY

스크롤할 때 특정 지점에 도달하면 그 위치에 붙어서 따라옵니다.

스크롤 테스트용 긴 영역

배지 (absolute)

3

알림 개수를 표시하는 배지를 absolute로 구현합니다.
