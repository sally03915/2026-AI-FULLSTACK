문서 기반 RAG AI 챗봇 (Thymeleaf + RestClient)

💡 RAG 챗봇 테스트 추천 질문 리스트
PDF를 생성하여 챗봇에 업로드한 후, 아래 질문들을 입력해 보며 RestClient를 통한 LLM 연동과 Vector DB 검색이 잘 작동하는지 검증해 보세요.

1. 숫자 및 금액 검증: "사내 도서 구입비 지원 한도는 얼마인가요?"	
2. 조건부 정보 검증: "개발자에게 지급되는 노트북 장비 종류는 무엇인가요?"
3. 고유 명사 검증: "2026년 상반기 핵심 프로젝트의 코드명은 무엇인가요?"
4. 문서에 없는 내용 검증 (할루시네이션 체크): "회사 근처 맛집을 추천해줘." 
(결과: 문서에 없는 내용이므로 "문서에서 관련 내용을 찾을 수 없습니다"라고 답변해야 정상입니다.)
※ 이미 문서를 업로드하셨거나 프로젝트 내 고정 문서(company.pdf)를 사용하려면 선택하지 않아도 됩니다.
